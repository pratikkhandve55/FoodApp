## 📖 About The Project

FoodApp is a social impact web application developed with the vision of reducing food wastage and helping people in need.

The platform allows users to:

✅ Donate excess food  
✅ Request food collection  
✅ Connect with volunteers  
✅ Contact the organization easily  
✅ Support hunger reduction initiatives

The project was originally created for a competition with a mission to **save lives by saving food**.
